Public Data Notice

The Open Cost Study Exchange is a public data project. Submitted files and submission records may be published in the public GitHub repository and on the public website.

Do not submit student-level records, student names, student IDs, employee IDs, Social Security numbers, dates of birth, protected personal information, or any data that your institution is not authorized to release publicly.

By submitting data, the submitting institution confirms that the data is approved for public release.

Submit Public Cost Study Data

Institutions may submit public data in either of two ways. Use email if you want the simplest process. Use GitHub Issues if you want a public submission record.

Option 1: Submit by Email

Email your completed public submission file and any notes. This is the easiest option for most institutions.

Please include the institution name, reporting year, contact person, and confirmation that the submitted file is approved for public release.

Email Public Submission

Option 2: Submit through GitHub

Create a public GitHub Issue using the structured submission form. This creates a transparent public record of the submission.

The issue should include the institution name, reporting year, data file link, contact information, and public-release confirmation.
