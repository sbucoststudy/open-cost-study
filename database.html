Open Cost Study Public Database

This page provides public benchmark files generated from public institutional submissions. The processed files include instructional cost per faculty member, cost per credit hour, percentile benchmarks, and data quality information.

Public Data Notice

All files linked from this page are intended for public use. Institutions should not submit student-level records, student names, student IDs, employee IDs, Social Security numbers, dates of birth, protected personal information, or any data that is not approved for public release.

Processed Public Database Files

These files are generated automatically from public raw submissions when the GitHub Actions database workflow runs.

  • Benchmark Database CSV — aggregate benchmark statistics by year, CIP code, and Carnegie classification
  • Public Institution Metrics CSV — public institution-level calculated metrics
  • Data Quality Report CSV — row counts and validation summary by source file
  • Benchmark Database JSON — machine-readable benchmark database used by the report generator
  • Public Institution Metrics JSON — machine-readable institution metrics used for percentile calculations
